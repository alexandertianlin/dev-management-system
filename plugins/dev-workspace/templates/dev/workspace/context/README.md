# Context

Store discoveries, codebase structure snapshots, and contextual information here.

## Files

| File | Purpose |
|------|---------|
| `tree.md` | Auto-generated directory structure |
| `discover.md` | Codebase discoveries and patterns found |

## How to Use

- Read `tree.md` at session start to understand project structure
- Use `dev-workspace tree` to regenerate the tree after structural changes
- Record important discoveries in `.md` files with clear filenames
